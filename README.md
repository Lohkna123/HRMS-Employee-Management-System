# HRMS-Employee-Management-System
Developed a web-based Human Resource Management System (HRMS) to efficiently manage employee records and HR operations. The system enables secure employee data management, real-time access to employee information, and reduces manual paperwork through an easy-to-use web interface.
 # Features
Employee Login
Admin Dashboard
Employee Record Management
Add, Edit & Delete Employees
Search Employees
Responsive User Interface
Secure Authentication
# Tech Stack
HTML5
CSS3
JavaScript
PHP
MySQL
